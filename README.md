# Como rodar:

1º)  Clone o repositório
```
git clone https://github.com/davimqz/Skyfall-Blitz.git
```

2º)  Entre na pasta do repositório
```
cd Skyfall-Blitz/
```

3º)  Compile os arquivos
```
make all
```

4º)  Rode o executável do jogo
```
make run
```
